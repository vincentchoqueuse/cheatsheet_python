Terminal Commands#

General Commands#

Listing Directory#

This command outputs the list of files in the current folder.

$ ls

Create Directory#

This command creates a new folder called my_folder in the the current folder.

$ mkdir my_folder

Change Directory#

This command changes the current folder to the folder my_folder.

$ cd my_folder

Move to Parent Folder#

This command changes the current folder to the parent folder.

$ cd ..

Python Programming#

Run a Python script#

This command runs the python script hello_world.py.

$ python hello_world.py

Install libraries#

This command installs the library flask.

$ pip install flask
