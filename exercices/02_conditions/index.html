Mathematical Programs¶

Exercice 1¶

Write a Python program that finds and displays the maximum of a list composed of three numbers. For example, if the content of the list is given by my_numbers = [19, 21, 5], the programs must display the number 21.

Specifications

  • Program Name: chap2_ex1.py

Exercice 2¶

Write a Python program that checks if a number value can be divided by 5 and by 7 and displays a message to the user using the print function.

Specifications

  • Program Name: chap2_ex2.py

  • Input: an integer value (must be provided by the user using input function)

Exercice 3¶

Write a Python program that randomly generates a number between 1 to 6 (included) using the randint() function and compares the generated number with a number guess specified by the user. If the number is equal to guess, the program must display the following message Well done, Bro !, else it must display the message No Luck, loser.

Specifications

  • Program Name: chap2_ex3.py

  • Library: random (see documentation)

  • Input: an integer guess (must be provided by the user using input function)

Exercice 5¶

Write a Python program that computes a price reduction. The initial price is stored in a variable price. The reduction price is then computed from the customer age.

  • When age < 12, the program must apply a reduction of 50%,

  • When age < 18, the program must apply a reduction of 25%,

  • When age >= 18, the program must not apply any reduction.

For example if price = 100 and age = 16, the program must display the value 75.

Specifications

  • Program Name: chap2_ex5.py

  • Input: an integer age and a float price (must be provided by the user using input function)

Exercice 6¶

Let us consider the following second order polynomial

\[p(x) = a_2x^2+a_1x +a_0.\]

The polynomial coefficients are assumed stored in a list equal to coefs = [1, -0.2, 0.4]. Write a python program that computes and displays the value of the roots of \(p(x)\).

Specifications

  • Program Name: chap2_ex6.py

  • Input: a float x (must be provided by the user using input function)

Turtle Drawings¶

Exercice 7¶

Write a Python program that draws a shape with L sides using Turtle.

  • if L is strictly lower to 2, the program does not draw any shape,

  • if L is equal to 2, the program must draw a triangle with three sides with equal length of 100,

  • if L is strictly greater then 2, the program must draw a square with side length 100.

Specifications

  • Program Name: chap2_ex7.py

  • Library: turtle (see documentation)

  • Input: an integer L (must be provided by the user using input function)

Exercice 8¶

Write a Python program that compares a user string input with the password value "secret145".

  • If the user string is equal to "secret145", the program must draw a green rectangle using turtle,

  • If the user string is not equal to the password, the program must display a red rectangle using turtle.

The user string must be specified by the user during runtime using the input built-in function.

Specifications

  • Program Name: chap2_ex8.py

  • Library: turtle (see documentation)

  • Input: a string code (must be provided by the user using input function)
