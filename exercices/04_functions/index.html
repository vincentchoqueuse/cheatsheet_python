Mathematical Programs¶

Exercice 1¶

Create a Python function that computes the minimum of a list of numbers.

Prototype

name:

minimum

inputs:

number_list (a list of float)

outputs:

a number (float)

Test

After the instruction output = minimum([19, 15, 5, 11]), the value of output must be 5.

Exercice 2¶

Create a Python function that computes the mean (i.e. the average) of a list of numbers.

Prototype

name:

mean

inputs:

number_list (a list of float)

outputs:

a number (float)

Test

After the instruction output = mean([19, 15, 5, 11]), the value of output must be 12.5.

Exercice 3¶

Create a Python function that compute the running average of a list of numbers. A running average is computed as follows

\[y[n] = \frac{x[n] + x[n-1]}{2}\]

where \(x[n]\) and \(y[n]\) for \(n=0,1,...,N-1\) corresponds to the input and output arguments of the function. For the first value \(n=0\), it is assumed that \(x[-1]=0\).

Prototype

name:

fir

inputs:

number_list (a list of float)

outputs:

output_list (a list of float)

Test

After the instruction output = fir([19, 15, 5, 11]), the value of output must be [9.5, 17, 10, 8]

Exercice 4¶

Create a function power that computes the value of \(y = x^k\) without using the standard function pow of the Python library.

To compute the mathematical power, it is recommended to use a loop.

Prototype

name:

power

inputs:

  • x (a float)

  • k (an integer)

outputs:

y (a float)

Test

After the instruction output = power(2, 3), the value of output must be 8.

Exercice 6¶

Create a Python function that draw a sinewave with the module and matplotlib modules using the mathematical expression

\[x[n] = a\sin(2\pi f n/Fe)\]

for \(n=0,1,2,..., N-1\)

Prototype

name:

show_sine

inputs:

  • f (a float)

  • a (a float)

  • Fe (a float)

  • N (an integer)

outputs:

None (the function only plots a figure)

Note

To evaluate the mathematical expression, it is recommended to use the arange and sin function of the Numpy module.
