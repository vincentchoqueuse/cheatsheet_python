Mathematical Programs#

Exercice 1#

Let us consider a python list composed of 4 numbers. The list is given by marks = [10, 19, 5, 4]. Write a Python program that computes and displays the mean of 4 numbers stored in the list marks.

Specifications

  • Program Name: 01_math.py

Exercice 2#

Let us consider the following second order polynomial

\[p(x) = a_2x^2+a_1x +a_0.\]

The polynomial coefficients are stored in the list coefs = [1, -0.2, 0.4]. Write a Python program that evaluates and displays the value of \(p(x)\). To evaluate each term, it is recommended to use the mathematical operator ** that raises the number on the left to the power of the exponent of the right (for example 2**3 corresponds to \(2^3\).)

Specifications

  • Program Name: 02_math.py

  • Input: a float x (must be provided by the user using input function)

Exercice 3#

Write a Python program that randomly displays a number between 1 and 6 (included) using the randint function of the random module.

Specifications

  • Program Name: 03_math.py

  • Library: random (see documentation)

Exercice 4#

Write a Python program that computes the volume of a sphere from its radius i.e.

\[V = \frac{4}{3}\pi r^3\]

Specifications

  • Program Name: 04_math.py

  • Library: math (see documentation)

  • Input: a float radius (must be provided by the user using input function)
