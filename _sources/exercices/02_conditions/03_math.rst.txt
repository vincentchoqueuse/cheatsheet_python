Exercice 3
++++++++++

Write a Python program that randomly generates a :code:`number` between 1 to 6 (included) using the :code:`randint()` function and compares the generated number with a number :code:`guess` specified by the user.
If the :code:`number` is equal to :code:`guess`, the program must display the following message :code:`Well done, Bro !`, else it must display the message :code:`No Luck, loser`.

**Specifications**

* Program Name: :code:`chap2_ex3.py`
* Library: :code:`random` (see `documentation <https://docs.python.org/3/library/random.html>`__)
* Input: an integer :code:`guess` (must be provided by the user using :code:`input` function)
