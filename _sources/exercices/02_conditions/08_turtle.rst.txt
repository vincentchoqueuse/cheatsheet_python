
Exercice 8
++++++++++

Write a Python program that compares a user string input with the password value :code:`"secret145"`. 

* If the user string is equal to :code:`"secret145"`, the program must draw a green rectangle using turtle,
* If the user string is not equal to the password, the program must display a red rectangle using turtle.

The user string must be specified by the user during runtime using the :code:`input` built-in function.

**Specifications**

* Program Name: :code:`chap2_ex8.py`
* Library: :code:`turtle` (see `documentation <https://docs.python.org/3/library/turtle.html>`__)
* Input: a string :code:`code` (must be provided by the user using :code:`input` function)
