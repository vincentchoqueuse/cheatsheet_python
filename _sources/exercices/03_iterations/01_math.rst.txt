Exercice 1
++++++++++

Write a Python program that computes the sum of the `geometric serie <https://en.wikipedia.org/wiki/Geometric_series>`_

.. math ::

    s_n = \sum_{n=0}^{N} q^n = 1 + q + q^2 + \cdots + q^N


**Specifications**

* Program Name: :code:`chap3_ex1.py`
* Input: an integer :code:`N` and a float :code:`q` (must be provided by the user using :code:`input` function)
