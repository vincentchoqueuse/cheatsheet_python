Exercice 1
++++++++++

Create a Python function that computes the minimum of a list of numbers.

**Prototype**

    :name: `minimum`
    :inputs: number_list (a list of float)
    :outputs: a number (float)

**Test**

After the instruction :code:`output = minimum([19, 15, 5, 11])`, the value of output must be 5.
